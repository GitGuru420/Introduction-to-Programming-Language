Topic: Basic Syntax, Variable & Data-types

Q1. What is Computer Programming?
Ans: The process of giving instructions to a computer to perform specific tasks.

Q2. What is C Programming?
Ans: Programming using C language is called C programming.

Uses of C Language:
The language is used to develop operating system, database, embedded system, graphical user interphase(GUI) and games. 

Q3. What is Header File?
Ans: Header files are basically files with .h extension that describe the function of various library functions.

Some Header File:
stdio.h : input and output function
stdlib.h : memory allocation and program termination
string.h : string manipulation
math.h : mathematical function
time.h : time and date function
